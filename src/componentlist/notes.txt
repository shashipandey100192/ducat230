git 
=======================
1) login account
2) git remote -v  ( check git connection)
3)  create Repositories 
4) git add --all ( all file add in git)
5) git remote add origin <repo path>   ( remote add )
6) git commit -m "any message"
7) git push origin <name of branch>
8) git clone <path>
9) npm install after clone or if install any 
10) git pull origin <branch name>
